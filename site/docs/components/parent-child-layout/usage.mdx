### Using the component

### When to use

Use Parent Child Layout when you want your layout to be split into two regions in a responsive manner.

### When not to use

Don't use Parent Child Layout when you want your layout to flow in a single dimension. In this case we recommend you having a look at [Stack Layout](?path=/docs/documentation-core-layout-stack-layout--docs), [Flow Layout](?path=/docs/documentation-core-layout-flow-layout--docs) or [Flex Layout](?path=/docs/documentation-core-layout-flex-layout--docs).

### Import

To import `ParentChildLayout` and `ParentChildItem` from the lab Salt package, use:

```js
import { ParentChildLayout, ParentChildItem } from "@salt-ds/lab";
```
