### Keyboard interactions

<KeyboardControls>
<KeyboardControl keyOrCombos="Enter / Space">

- If focus is on the trigger element, this action activates the element and opens drawer. Focus remains within the drawer.
- If focus is on close button, this action closes the drawer. Focus returns to trigger element.
- If focus is on a content element configured to close the drawer when activated, such as a _Submit_ button, this action closes the drawer.

</KeyboardControl>
<KeyboardControl keyOrCombos="Tab">

- Tab moves focus through focusable content elements in the drawer. Focus does not leave the drawer.

</KeyboardControl>
<KeyboardControl keyOrCombos="Shift + Tab">

- This action moves focus to last focusable content element in the drawer. Focus does not leave the drawer.

</KeyboardControl>
<KeyboardControl keyOrCombos="Escape">

- Escape closes the drawer and returns focus to trigger element.

</KeyboardControl>
</KeyboardControls>
