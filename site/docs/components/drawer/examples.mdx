### Default

By default, a drawer anchors to the `left` edge of the screen and has `variant="primary"`.
Set `variant="secondary"` for secondary drawer.

</LivePreview>

<LivePreview componentName="drawer" exampleName="Position" >

### Position

You can place drawer on the `left`, `right`, `top` and `bottom` edges using `position` prop.

</LivePreview>

<LivePreview componentName="drawer" exampleName="RightUseCase" >

### Right drawer use case

Consider using a right-anchored drawer to show supporting information related to the current context, such as recent events, notifications or quick settings.

</LivePreview>

<LivePreview componentName="drawer" exampleName="BottomUseCase" >

### Bottom drawer use case

Anchor a drawer to the bottom edge of the screen when using a master-detail layout with content, such as graphs, images or comparisons that requires a wider horizontal format.

</LivePreview>

### Optional close button

This example has been configured to close on `Submit` click, instead of closing with drawer's optional close button. When present, close button always appear on the top right corner of drawer.
